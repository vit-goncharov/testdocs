Functions

RegKey:create (parentKeyHandle, keyName[, access=KEY_ALL_ACCESS[, options=REG_OPTION_NON_VOLATILE]])
Create or open if exists a key

Parameters:

  • parentKeyHandle handle of the parent key or one of predefined keys: "HKEY_CLASSES_ROOT", "HKEY_CURRENT_USER", "HKEY_LOCAL_MACHINE", "HKEY_USERS"
  • keyName name of the target key
  • access access attributes to the key (default KEY_ALL_ACCESS)
  • options (default REG_OPTION_NON_VOLATILE)
RegKey:open (parentKeyHandle, keyName[, access=KEY_ALL_ACCESS[, options=0]])
Open existing key

Parameters:

  • parentKeyHandle handle of the parent key or one of predefined keys: "HKEY_CLASSES_ROOT", "HKEY_CURRENT_USER", "HKEY_LOCAL_MACHINE", "HKEY_USERS"
  • keyName name of the target key
  • access access attributes to the key (default KEY_ALL_ACCESS)
  • options (default 0)
RegKey:getVal (valueName[, subKeyName=nil[, size[, flags=RRF_RT_ANY]]])
Get value of the key

Parameters:

  • valueName name of the value
  • subKeyName subkey of the key to get value from (default nil)
  • size assumed size of the value, if nil the function detemines it itself but it costs additional API call (optional)
  • flags operation flags (default RRF_RT_ANY)
RegKey:setVal (valueName, valueType, data, size[, subKeyName])
Set value of the key

Parameters:

  • valueName name of the value
  • valueType type of the value
  • data data to write
  • size size of the data
  • subKeyName name of the subkey (optional)

Returns:

    0 on success
