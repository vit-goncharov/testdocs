File.open (path, disp[, access=GENERIC_READWRITE[, attr=FILE_ATTRIBUTE_NORMAL[, share=FILE_SHARE_ALL]]])
Create or open a file

  • disp int
  • access int to the file (default GENERIC_READWRITE)
  • attr int fileattributes (default FILE_ATTRIBUTE_NORMAL)
  • share int sharing options (default FILE_SHARE_ALL)
